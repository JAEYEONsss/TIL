# 마크다운 링크 삽입
## 1. URL 직접입력
* 삽입하고자 하는 링크에 http 혹은 https를 입력하면 자동으로 링크가 삽입된다.
```
https://hello.com
```

## 2. 텍스트에 링크 삽입
```
[눌러보세요](https://hello.com)
```

## 3. 링크에 툴팁 삽입
```
[눌러보세요](https://hello.com "테스트입니다")
```

## 4. 이미지에 링크 삽입
* 이미지에 링크를 삽입하려면, [![이미지설명(이미지 소스 URL)](링크URL)]과 같은 형식으로 입력
```
[![이미지설명(https://hello.com/hello.png)](bye.com)]
```
